1st Project: Password Storer

This is a project to imitate popular password-storing applications such as 1Password©. The idea is that a single password can be used to control access to a list of all the passwords associated with an account. This necessitates a single password that is strong enough to prevent access to unwanted parties but simultaneously memorable enough for the user to remember itself. The project therefore had several requirements:

  • Have one long, unique password to gain access to other passwords
  • Be accessible and easy to use
  • Have extra functionality such as being able to search
